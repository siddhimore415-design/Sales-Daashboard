Sales Dashboard (Power BI)
Overview

This project is an interactive Sales Dashboard created using Power BI to analyze sales performance, profit trends, customer behavior, and product insights from 2020–2025.

The dashboard provides a business overview through KPIs, charts, maps, and filters that help users make data-driven decisions.

Data Visualization Techniques
Dashboard Components
1. Profit by Category

Analyzes profitability across different product categories.

2. Quantity by Sub-Category

Displays product demand using donut chart visualization.

3. Top Customers

Highlights customers generating maximum sales.

4. Sales by State

Provides geographical sales insights using map visuals.

5. KPI Cards

Summarizes important business metrics.

Filters Used
Year Filter
State Filter
Order ID Filter

Key Insights
Office Supplies and Furniture generated the highest profit.
A few customers contribute a large share of total sales.
Sales performance varies significantly by region.
Product sub-categories show different demand patterns.
